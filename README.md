# tjrides

## a project by Tanmai Kalisipudi and Ishaan Sharma

---

🚙 An all-in-one platform for students to rideshare to school – all for free.

To reproduce this code, go into each of the submodule folders (frontend and backend) and clone them. The instructions for running the code is located in their respective README's.

After starting both servers in the backend and frontend using `npm start` in both directories in the terminal, the app should work! Open http://localhost:3000/ and start using it.
